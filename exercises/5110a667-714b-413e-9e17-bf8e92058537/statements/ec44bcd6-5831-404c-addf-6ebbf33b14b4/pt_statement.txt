<p>A declaração <a href=”https://www.postgresql.org/docs/14/sql-update.html” target=”_blank”><code>UPDATE</code></a> permite que você para modificar uma série de valores de determinados registros nas tabelas do banco de dados.</p><h3>Sintaxe</h3><p>A maneira mais simples de usar a instrução <code>UPDATE</code> é a seguinte sintaxe :</p><blockquote><pre><code> UPDATE table_name <br /> SET column_name = { expressão | DEFAULT } [, ...]<br /> [ WHERE condition ];</code></pre></blockquote><p>Onde <code>table_name</code> será o nome da tabela na qual você deseja modificar os dados. Você pode especificar os nomes de coluna desejados na tabela indicada. Cada coluna especificada deve ser associada ao novo valor usando o sinal =. Cada par <code>column_name = expressão</code> deve ser separado do próximo usando vírgulas (,).</p><p>A cláusula <code>WHERE</code> seguida pela condição é opcional (como pretendido pelos colchetes). Se indicado, a atualização dos dados afetará apenas os registros que atendam à condição. Portanto, lembre-se de que se você não especificar a cláusula <code>WHERE</code>, as alterações afetarão todos os registros da tabela.</p><h3>Exemplo</h3><p>Vamos incrementar o salário dos funcionários, cujo cargo é <code>'IT_PROG'</code>, 20%.</p><blockquote><pre><code>BEGIN;<br />SELECT funcionário_id, salário FROM funcionários WHERE JOB_ID = 'IT_PROG' ORDER BY employee_id;<br /><br /><strong>UPDATE </strong>funcionários<br /><strong>SET </strong>SALARY = SALARY * 1.2<br /><strong> WHERE < /strong>JOB_ID = 'IT_PROG';<br />-- UPDATE 5<br /><br />SELECT employee_id, salário FROM employees WHERE JOB_ID = 'IT_PROG' ORDER BY employee_id;<br />ROLLBACK;< /code ></pre></blockquote><h3>Exercício</h3><p>Modificar o departamento cujo nome é 'TI' colocando como novo <code>MANAGER_ID</code> o funcionário com identificador 104</p >
